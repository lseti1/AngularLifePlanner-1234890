Welcome to Life Planner

Plan your days, week or months ahead and see what's coming up.

Features include:

  • Add, edit, and delete plans
  • Side panel with easy access to various key dates
  • Add multiple plans to each day and track whether they're complete
  • and more!

Note: This is a demo application. Data is saved in your browser's local storage and can be cleared anytime via the settings. Not all features may work as intended, and this is not a full-fledged planning platform. Best Viewed in Full Screen on a PC.

(Do not store any personal information while using this app)
